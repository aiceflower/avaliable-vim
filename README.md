# avaliable-vim
简单有用的便携式vim插件配置

### 1.安装

git clone https://github.com/aiceflower/avaliable-vim.git

复制 avaliable-vim/目录下所有文件到用户家目录

在 .bash_profile文件中添加如下内容
  if [ -f ~/.bashrc.before ]; then
    . ~/.bashrc.before                                                                                                                  
  fi

重新加载配置文件 . .bash_profile
